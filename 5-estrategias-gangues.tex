\section{Estratégias de Gangues Criminosas}

A análise de mais de 16.000 ataques identifica dois perfis principais.

\subsection{Gangues Generalistas}

Atacam múltiplos setores e utilizam técnicas avançadas:
\begin{itemize}
  \item LockBit;
  \item BlackCat/ALPHV;
  \item Cl0p.
\end{itemize}

Características marcantes:
\begin{itemize}
  \item automação;
  \item exploração em larga escala;
  \item criptografia rápida;
  \item alta taxa de sucesso.
\end{itemize}

\subsection{Gangues Especializadas}

Focam em setores específicos:
\begin{itemize}
  \item saúde;
  \item finanças;
  \item manufatura.
\end{itemize}

Usam técnicas mais simples:
\begin{itemize}
  \item spear-phishing;
  \item credenciais vazadas.
\end{itemize}
